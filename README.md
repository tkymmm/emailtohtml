# emailtohtml
## EML → HTML 変換ツール（v1.2）

Windows 向けの **EML / Outlook メールを HTML に変換するツール**です。  
GUI で複数ファイルをまとめて変換でき、ドラッグ＆ドロップにも対応しています。

---

## 🚀 主な機能

### ✔ EML を HTML に変換
- メール本文（HTML / TEXT）を正しく抽出
- 文字化けしにくい日本語向けデコード処理
- `<meta charset="UTF-8">` を自動挿入して表示崩れを防止

### ✔ 添付ファイルの自動保存
- `メール名_files/` フォルダにまとめて保存
- 元のファイル名を維持

### ✔ 外部画像（http/https）の保存（v1.2）
- `<img src="https://...">` の画像をローカルに保存
- 元のファイル名を維持
- 重複時は `banner(1).jpg` のように連番を付与
- HTML 内の参照も自動でローカルパスに書き換え
- GUI では ON/OFF 可能  
- D&D では常に ON

### ✔ GUI で簡単操作
- 複数ファイルを一覧に追加
- 「すべて選択」「選択解除」「選択項目を削除」「クリア」
- 出力先フォルダの指定
- 変換ボタンを押すだけ

### ✔ exe にドラッグ＆ドロップで一括変換
- 複数の EML をまとめて変換
- 完了ダイアログ付き

## 🔧 使い方

### 1. GUI で使う場合
1. `EML 変換ツール.exe` を起動  
2. 「ファイルを選択」で EML を追加  
3. 必要なら出力先フォルダを指定  
4. 「変換する」を押すだけ  

### 2. ドラッグ＆ドロップで使う場合
`EML 変換ツール.exe` に EML をまとめてドロップすると  
自動で変換が始まります。
